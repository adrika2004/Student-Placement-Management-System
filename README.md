# Student Placement Management System

## Overview

Student Placement Management System is a full-stack web application designed to simplify and automate the campus placement process. The platform enables students to explore placement opportunities, manage profiles, upload resumes, and track applications while providing administrators with tools to manage recruitment activities efficiently.

## Key Features

### Student Module

* Student registration and secure login
* Profile management and resume upload
* Browse available placement opportunities
* Apply for job openings
* Track application status

### Administrator Module

* Manage student records
* Create and manage placement drives
* Review applications
* Schedule interviews
* Monitor placement activities

### Analytics & Monitoring

* Placement statistics dashboard
* Application tracking
* Recruitment activity monitoring

## System Architecture

The application follows a client-server architecture where the React frontend communicates with the Express backend through REST APIs. User authentication is handled using JWT tokens, while MongoDB stores application data including student profiles, company details, and placement records.
